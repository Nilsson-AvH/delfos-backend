# Project Delfos

Delfos es un proyecto de backend que se encarga de gestionar los empleados de la empresa Delfos.

## Caracteristicas desarrolladas
**Entidad: User**
- [X] Registro de usuarios
- [X] Login de usuarios
- [X] Obtener Listado de usuarios
- [ ] Obtener Listado de un usuario por id.
- [ ] Actualizacion de usuarios por id.
- [X] Eliminacion de usuarios por id. 

## Tecnologias

- Node.js
- Express
- Mongoose
- MongoDB
- dotenv
- bcryptjs
- jsonwebtoken
- cors
- helmet
- morgan
